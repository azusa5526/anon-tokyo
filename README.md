# Anon Tokyo

[連結](https://anon-tokyo.pages.dev/)

## 簡介

在這裡，你可以搜尋並加入喜歡的台詞到素材庫，排列、編修，最後製作成你專屬的動圖。

## 功能

- **搜尋片段：**
  - 以台詞搜尋相關片段

- **合成動圖：**
  - 將多個片段剪接成動圖

## 使用技術工具

- Vue 3
- Vite 6
- TypeScript
- Tailwind CSS
- Vuedraggable
- Vue Virtual Scroller
